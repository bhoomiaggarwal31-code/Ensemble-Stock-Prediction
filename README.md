## Project Overview

This project predicts stock closing prices using Ensemble Machine Learning techniques. Multiple regression models are trained and combined using a Voting Regressor to improve prediction performance.

---

## Features

- Stock Price Prediction
- Ensemble Learning
- Linear Regression
- Random Forest
- Gradient Boosting
- Voting Regressor
- Exploratory Data Analysis
- Streamlit Web Application

---

## Dataset

Features:

- Open
- High
- Low
- Volume

Target:

- Close Price

---

## Models Used

| Model | R² Score |
|--------|----------|
| Linear Regression | 0.999937 |
| Random Forest | 0.999879 |
| Gradient Boosting | 0.999829 |
| Voting Regressor | 0.999905 |

The Voting Regressor is used for deployment to demonstrate ensemble learning.
